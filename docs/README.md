# Repository `docs/` (placeholder)

This directory is reserved for **repository‑root, system‑level documentation**: overall architecture, cross‑subsystem design, and technical decisions that apply to **KWeaver Core as a whole**.

- **Module‑scoped** design and decisions stay under each subsystem’s own `docs/` (for example under `adp/`, `decision-agent/`, `infra/`, and so on).
- **User manuals** live under `help/{en,zh}/manual/`.
- **Cookbooks** live under `help/{en,zh}/cookbook/`.

For the full layout rules, see **[CONTRIBUTING](../rules/CONTRIBUTING.md)** — section **DOCUMENTATION**. Chinese: **[CONTRIBUTING.zh](../rules/CONTRIBUTING.zh.md)**.
