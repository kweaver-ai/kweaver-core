# text_processor_skill

这是一个用于测试的文本处理技能包。

## 功能说明
提供简单的文本处理功能。

## 输入参数
- text: 输入文本 (string)

## 输出结果
- processed_text: 处理后的文本 (string)

## 使用示例
```json
{
  "text": "Hello World"
}
```

返回：
```json
{
  "processed_text": "hello world"
}
```
